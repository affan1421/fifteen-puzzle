# Fifteen puzzle

Simple and interactive fifteen puzzle game. client side is completely created with reactJs.

### Features

- Using ReactJs
- Added interactive functionality like vibrate and sound on moves
- Uses mantine as ui library.
- Customizable layout and code.

### To use and test
As a developer who is entusiastic, and wants to learn more. you can always run this code and see the code running by cloning this repository 

```
  git clone https://github.com/remintroy/React-FifteenPuzzle.git
```

Install dependencies 

```bash 
   npm i 
```
Run code in development mode

```bash 
   npm run dev
```

Now you are all set and good to go.
